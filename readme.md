# Cartes de personnages (API Harry Potter)

Page web permettant d'afficher des cartes de personnages issues de l'API Harry Potter.  
Les données sont récupérées via fetch() puis affichées sous forme de cartes dynamiques en HTML/CSS/JS.

## Fonctionnalités
- Récupération des données via l'API
- Affichage dynamique des cartes
- Mise en page responsive
- Gestion des images manquantes

## Technologies utilisées
- HTML
- CSS
- JavaScript
